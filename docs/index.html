Quick Start

Get Running in Minutes

Three simple steps to your local AI setup.

Terminal 
# 1. Clone or download the repository
git clone https://github.com/irfan-ai/ai-stack.git
cd ai-stack

# 2. Run the lean setup (downloads models)
./scripts/setup-lean.sh

# 3. Start the full stack
./scripts/setup.sh

Choose Your Interface

Three Modes 
# Terminal Mode (~50MB RAM)
./scripts/local.sh

# Simple Web UI - No Docker (~200MB RAM)
./scripts/web-simple.sh

# Full WebUI with Docker (~2-3GB RAM)
./scripts/web.sh
Comparison

Choose Your Setup

Different configurations for different needs and hardware.

Setup RAM Usage Features Best For
Terminal Mode ~50MB CLI Chat, Ollama models Power users, quick tasks
Simple Web UI ~200MB Browser chat, model selection Lightweight browsing
Full Open WebUI ~2-3GB RAG, file uploads, cloud models Complete experience
